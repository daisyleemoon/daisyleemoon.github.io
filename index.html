WeHelp Bootcamp 問題回應

個人簡介

畢業於台北大學經濟學系。曾任資訊公司行政助理兩年，消防特考及消防員訓練一年，任職正式消防員一年半。因日夜班不固定輪值導致個人健康因素離職，近四年在自家早餐店協助經營。
擁有良好的抗壓能力與溝通能力，在高強度的消防員訓練中完成所有目標並取得七項證照，任職期間能與夥伴無誤的完成出勤任務，在不同的工作環境中，也能與各種類型的同事合作。

為什麼想成為前端、後端、或全端工程師？

主要有兩個原因，首先是自家早餐店受到 COVID-19 嚴重的影響，其次是身邊友人成功轉職成軟體工程師，一樣身為文組大學畢業生，這讓我開始思考自己是否有同樣的可能性。之後在網路社群找到了許多文組成功轉職的案例分享後，更堅定自己的決心。
學習軟體開發是對自我的一種投資，軟體工程師的需求在各種不同的產業都逐年增長，能夠擁有這方面的專長，我相信對自己的未來大有裨益。目前不設限於前後端工程師，希望能對於全端的技能自己都能有一些基礎的了解，以提升自己求職成功的可能性。

為了成為軟體工程師，曾經做過什麼努力？

為了確認自己是否有興趣成為軟體工程師，我在Coursera完成了下列1-4點的課程。

  1. Introduction to HTML5結業證書
  2. Introduction to CSS3結業證書
  3. Interactivity with JavaScript結業證書
  4. Advanced Styling with Responsive Design結業證書
  5. The Complete Web Developer in 2022: Zero to Mastery進度截圖

之後在Udemy 購買第 5 項課程開始自學，經歷兩個半月已完成 60% 的進度，預計 12 月可以全部完成。目前對於下列項目已有一定程度的瞭解：

  • HTML：tag用法
  • CSS：語法、Grid Layout
  • Bootstrap
  • JavaScript: Data Structure (Array and Object)、 Function、Loop
  • Git
  • React

如果參與這個計畫，會怎麼安排學習時間？

獲得家人的支持會全職參與學習營，時間安排為：
星期一至六，每日安排三個時段學習，一天共9小時一周共54小時。

每周一到六 9:00-12:00 14:00-18:00 14:00-18:00
週日 學習進度允許的情況下作為緩衝，用來休息充電或是惡補的時間

目標是安排規律的作息、維持良好的狀態、提升學習效率。

如果參與這個計畫，預期會碰到什麼困難？你打算怎麼解決它？

預期可能會遇到下列的困難：

  1. 沒有相關經驗的自己如何跟上學習營的步調
    2. 我會全職準備這個學習營，參考學習營的大綱預習，不會害怕在學習營中提出問題尋求幫助，也會適當尋求身邊友人的協助加速自己解決問題。
  2. 跨領域的挫折感
    3. 我會運用我成功完成消防員訓練的經歷來幫助我克服這部分，相似的部分為都有陌生的新事物還有時間上的壓力，很容易感到恐懼與挫折，此時我會反覆練習熟能生巧，調整良好的心態積極面對挫折，我相信挫折之處就是自己不足之處，跨越過後會使自己更加強大。

是否有想要加入的軟體公司？為什麼想加入該公司？

我想選擇專注軟體開發的公司做為我軟體工程師職涯的第一份工作，以熟悉開發流程、提升自己的技術能力，同時精進有興趣一端的能力，以利自己可以追求更好的薪資條件與更多發展的選擇。
其次累積工作經驗以及專業領域知識，追求進入有興趣的產業，例如：Web3.0相關技術的產業，因為期望每個人都可以使用更透明、自由、公平的網路，相關領域也正蓬勃發展，未來可以有更多應用期許自己也能有加入的一天。

想要對我們說的事情？

我希望能得到學習營的幫助：

  1. 縮短自己學習模索的時間
    2. 一個人在網路上搜尋雖然可以獲得解答但是有時候關鍵字用的不夠精確就要花上更多的時間得到解答，若有有經驗的人協助往往可以事半功倍，這點在友人的協助上獲得驗證，可惜他們也有正職我不能一直打擾他們。
  2. 明確的學習目標與方向
    3. 自學的過程中會感到困惑自己準備的方向是否正確，也對於所學的技能是否足夠了解而且達到標準感到懷疑，希望能藉由完成實作來證明自己有獨立完成實作專案的能力。
  3. 同儕的鼓勵
    4. 學習一直是很孤獨的一件事，但是能有同儕互相支持可以減少這一路上的孤獨感，也能一起討論彼此學習的問題互相激精進。
